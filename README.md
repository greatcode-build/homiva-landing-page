# ![Homiva Logo](icons/logo.svg) Homiva

A modern, responsive real estate landing page built with React, TypeScript, and Vite. Homiva showcases luxury properties, company information, and client testimonials with smooth animations and a clean design.

## ✨ Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Powered by Motion.js for engaging user interactions
- **Property Showcase**: Interactive carousel displaying luxury real estate projects
- **Client Testimonials**: Social proof from satisfied customers
- **Contact Integration**: Easy-to-use contact form with toast notifications
- **Modern UI**: Styled with Tailwind CSS for a professional look

## 🛠️ Tech Stack

- **Frontend**: React 19 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Motion.js
- **Notifications**: React Toastify
- **Linting**: ESLint with TypeScript support

## 🚀 Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd homiva-landing-page
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

To build the project for production:

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

To preview the production build locally:

```bash
npm run preview
```

## 📝 Usage

The landing page consists of several sections:

- **Header**: Hero section with navigation and call-to-action buttons
- **About**: Company information and key statistics
- **Projects**: Interactive carousel of completed real estate projects
- **Testimonials**: Client reviews and ratings
- **Contact**: Contact form for inquiries
- **Footer**: Additional links and company information
